Learning Animate.css

Bounce

The bounce animation makes an element look like it bounces upon page loading. This section bounces using the basic usage style i.e. it adds the animate__animated and animate__bounce classes to the element.

Another Bouncy Section

This section is made bouncy using the keyframes style of bouncing. Using @keyframes provides a greater level of flexibility and control over the animation' appearance. Plus it doesn't require you to refactor HTML code.
